# che-cosa-faremo-
group project for the transtextualites numeriques class in paris 8 with french and italian students.

new idea (friday):


We chose images from Le Bon Coin (France) and Subito (Italy).

Locations are Saint Senis in Le Bon Coin and Trecate in Subito.

The Italian students search in Le Bon Coin and the French students search in Subito.

We find 2 images each, and write something about it. We write text around the objects/images. External narrator talking about the object (story through objects).

Location of the images/objects is important.
